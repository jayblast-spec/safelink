# Contributing to SafeLink

Thank you for helping improve SafeLink. This project is open-source by design: ideas become stronger when contributors can improve product quality, code quality, security, accessibility, and commercial usefulness.

## Good First Contributions

- Improve the live workflow so it produces more useful Link intelligence outputs.
- Add tests for the core product logic.
- Improve mobile responsiveness and accessibility.
- Add source-backed integrations or export formats.
- Strengthen README examples and product documentation.

## Quality Bar

Every contribution should make the project more useful to SOC analysts, creators, teams, and families. Avoid generic UI, placeholder copy, or broad AI-wrapper claims.

## Local Setup

```bash
npm install
npm run dev
```
